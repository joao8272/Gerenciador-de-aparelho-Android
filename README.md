Projeto que gerencia funções de aparelhos android. Criado para fim institucional de corporação para bloquear acesso a recursos/aplicativos específicos para usuários sem senha de administrador em aparelhos corporativos, bloqueando tudo que o a administração considerar irrelevante para a utilização do aparelho por usuários não admnistradores. O bloqueio automatico pode ser ativo tanto em apps criticos de sistema que geralmente não é possivel serem bloqueados, como o app discador/telefone. Assim como é possivel bloquear diretamente sites não permitidos, ocasionanando caso algum usuário entre em um site bloqueado, acionando o bloqueio imediatamente com a menor interação que tiver com o site, e forçando a saida do mesmo. Resultando que, quando o usuário entrar no navegador novamente após a saida forçada de um site bloqueado, o navegador é rezetado para a tela de pesquisa inicial. Todos os desbloqueios com senha de admnistrador são temporareos, ou seja, se um usuário administrador tentar entrar em um app bloqueado e colocar a senha de acesso correta, ele vai conseguir utilizar o app temporareamente, mas no momento em que ele sair do app e tentar entrar de novo, vai pedir a senha novamente, para evitar que o dispositivo fique liberado para usuários sem permissão caso algum usuário administrador coloque a senha correta, o mesmo efeito acontece em sites bloqueados.
