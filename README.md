Projeto que gerencia funções de aparelhos Android. Criado para fins institucionais de uma corporação, com o objetivo de bloquear o acesso a recursos/aplicativos específicos para usuários sem senha de administrador em aparelhos corporativos, bloqueando tudo o que a administração considerar irrelevante para a utilização do aparelho por usuários não administradores.

O bloqueio automático pode ser ativado tanto em apps críticos do sistema que geralmente não podem ser bloqueados,como o app discador/telefone e todos os outros. Também é possível bloquear diretamente sites não permitidos. Caso algum usuário acesse um site bloqueado, o sistema aciona o bloqueio imediatamente, com a menor interação realizada no site, forçando a saída do mesmo.

Como resultado, quando o usuário entrar novamente no navegador após a saída forçada de um site bloqueado, o navegador será resetado para a tela de pesquisa inicial.

Todos os desbloqueios com senha de administrador são temporários. Ou seja, se um usuário administrador tentar entrar em um app bloqueado e inserir a senha correta, ele poderá utilizar o app temporariamente. No momento em que sair do app e tentar acessá-lo novamente, será solicitada a senha outra vez, evitando que o dispositivo permaneça liberado para usuários sem permissão caso um administrador tenha inserido a senha anteriormente. O mesmo efeito ocorre em sites bloqueados
